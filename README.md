## React后台管理系统

### 1 . 项目简介

1) 此项目是一个后台管理的 SPA 

2) 使用 React 全家桶 + Antd + Axios + ES6 + Webpack 等技术 

3) 采用模块化、组件化、工程化的模式开发

### 2 . 项目启动

开发环境运行

```javascript
npm start
```

生产环境运行

```
npm run build
```

测试环境

```
npm run test
```

### 3 . 技术选型

前后台数据展现/交互/组件化

```javascript
react react-router-dom antd redux
```

接口测试工具

```javascript
postman
```

模拟数据工具

```javascript
json-server mock-js
```

项目构建/工程化

```javascript
webpack creat-react-app eslint
```

富文本编辑器

```javascript
react-draft-wysiwyg draft-js draft-to-html
```

图标库

```javascript
echarts
```
